# 数据结构与算法速成计划 (C++版)

这是一个为期3周的"数据结构与算法速成计划"网页应用，专为希望快速掌握核心数据结构与算法知识的C++学习者设计。该应用提供了结构化的学习路径、每日任务安排、LeetCode练习题链接以及进度跟踪功能，帮助你高效备考或提升编程技能。

## 功能特点

- **结构化学习计划**：将数据结构与算法知识分为3周，每周聚焦不同主题，循序渐进
- **每日任务安排**：详细列出每天需要学习的内容、重点、参考书籍章节和对应的LeetCode练习题
- **进度跟踪**：可标记已完成的学习任务和编程题目，实时查看学习进度
- **个人记录**：提供进度跟踪表格，记录每日完成内容、难点和明日计划
- **数据持久化**：自动保存学习进度到本地存储，下次打开无需重新标记
- **Excel导出**：支持将学习计划导出为Excel文件，方便离线查看和打印
- **响应式设计**：适配不同屏幕尺寸，在电脑、平板和手机上均可使用

## 学习内容概览

### 第1周：基础数据结构
- 数组与链表
- 栈与队列
- 哈希表
- 字符串处理
- 排序与二分查找

### 第2周：树与高级算法
- 二叉树与递归
- 二叉搜索树（BST）
- 堆与优先队列
- 分治与贪心算法
- 动态规划（DP）基础

### 第3周：图与复杂算法
- 图的表示与遍历
- 最短路径算法
- 回溯算法
- 动态规划进阶
- 综合高频题练习

## 使用指南

1. **浏览学习计划**：点击顶部的"第1周"、"第2周"、"第3周"标签页浏览相应内容
2. **标记完成任务**：点击任务项或LeetCode链接可标记为已完成（显示删除线）
3. **跟踪每日进度**：在"进度跟踪"标签页中记录每日学习情况
4. **添加新记录**：点击"添加记录"按钮可新增一行进度记录
5. **导出计划**：点击"下载Excel表格"可将整个学习计划导出为Excel文件
6. **重置进度**：如需重新开始，可点击"重置进度"按钮清除所有记录

## 技术实现

- HTML5 语义化标签构建页面结构
- Tailwind CSS 实现响应式布局和美观样式
- JavaScript 实现交互功能和本地数据存储
- SheetJS 库实现Excel导出功能
- Font Awesome 提供图标支持

## 本地使用

只需将HTML文件下载到本地，用任意现代浏览器打开即可使用，无需额外安装依赖。所有数据会保存在浏览器的本地存储中，确保页面关闭后进度不会丢失。

## 注意事项

- LeetCode练习题链接会跳转到对应的题目页面，需要有LeetCode账号才能提交代码
- 本地存储的数据仅保存在当前浏览器中，更换浏览器或清除浏览器数据会导致进度丢失
- 建议定期导出Excel文件作为进度备份
